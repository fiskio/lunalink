References & Standards
========================

This page provides direct access to the official standards, guidelines, and 
specifications that govern the LunaLink implementation.

Primary Specifications
----------------------

*   **LSIS-AFS**: `Lunar Net Signal-In-Space Recommended Standard – Augmented Forward Signal (Vol-A) <https://github.com/fiskio/lunalink/blob/main/docs/references/lunanet-signal-in-space-recommended-standard-augmented-forward-signal-vol-a.pdf>`_ (LSIS V1.0, 29 Jan 2025).
*   **CCSDS 235.1**: `Space Communications Session Control <https://public.ccsds.org/Pubs/235x1b1.pdf>`_ (Draft/Current Version).

Engineering & Mission Assurance
-------------------------------

*   **NASA Class A**: `NASA-STD-8739.8B: Software Assurance and Software Engineering <https://standards.nasa.gov/standard/NASA/NASA-STD-87398>`_.
*   **ESA ECSS**: `ECSS-E-ST-40C: Software Engineering Standard <https://ecss.nl/standard/ecss-e-st-40c-software-engineering/>`_.
*   **ESA ECSS-Q**: `ECSS-Q-ST-80C: Software Product Assurance <https://ecss.nl/standard/ecss-q-st-80c-rev-1-software-product-assurance/>`_.
*   **MISRA C++:2023**: Guidelines for the use of the C++ language in critical systems.

Python & Implementation Guidelines
----------------------------------

*   **ESA-GS-0871**: `ESA Python Coding Guidelines <https://github.com/fiskio/lunalink/blob/main/docs/references/esa_python_guidelines.pdf>`_ (PE-RS-ESA-GS-0871).
*   **ESA-GS-0872**: ESA Guidelines for Python development (PE-TN-ESA-GS-0872).
*   **LSIS Annex 2**: `LDPC Submatrices CSV README <https://github.com/fiskio/lunalink/blob/main/docs/references/002_LNIS-AD1-Vol-A-Annex2-LDPC-Submatrices-CSV-README.pdf>`_.
*   **LSIS Annex 3**: `PRN Spreading Codes README <https://github.com/fiskio/lunalink/blob/main/docs/references/005_LNIS-AD1-Vol-A-Annex3-PRN-Spreading-Codes-README.pdf>`_.
